图像对应的公式为$$y = ((x-1+s)\%i) + 1$$, 相当于向左移动了s个单位。（左加右减，上加下减）

s是如何计算的呢 $$s = (m-1)\%i + 1$$

带入上一个公式得到
$$前号 = (后号+(m-1)\%i)\%i+1$$

可以再次化简，得到公式
$$前号 = (后号+m-1)\%i)+1$$
