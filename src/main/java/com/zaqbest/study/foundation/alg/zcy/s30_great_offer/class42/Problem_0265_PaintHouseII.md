解法1：
二维表dp,dp[i][j]指定是第i个房子，必须是j颜色的最小代价
时间复杂度 O(N^2)

解法1-进阶版：
由于解法1存在枚举行为，可以修改为线段树方式
时间复杂度O(N*logN)

解法2：
i+1只需要知道4个信息，
i行的**最优**代价，i行**最优**代价的颜色；
i行的**次优**代价，i行**次优**代价的颜色；
